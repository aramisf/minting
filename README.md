# Minting

Yet another Ruby library for dealing with money and currency.

Work in progress, please wait release 1.0.0

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'minting'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install minting

## Usage

There are three main classes:
- Money
- Currency
- Mint
